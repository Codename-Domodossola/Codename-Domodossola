# Examples

This directory contains illustrative deployment material.

Content in this directory is non-normative and may include:
- deployment requirements
- architecture sketches
- implementation notes
- experimental configurations
- integration experiments

## Privacy and Contribution Policy

Real-world deployment configurations are private by default and are not
expected to be shared publicly.

Examples in this directory are voluntarily contributed, anonymised
representations of real deployments. Contributors are responsible for
ensuring that identifying information, site-specific data, cryptographic
material, and any other sensitive content have been removed before
submission.

See `CONTRIBUTING` for the full submission policy.
